# 吞嚥復健互動網頁

讓患者在家也能依照指示進行基本居家復健的互動式網頁。

## 功能特色

- 📋 **互動式復健動作指引** - 動畫示範每個復健動作
- 📊 **復健進度追蹤** - 記錄每日復健情況
- 🎥 **影片教學** - 各類復健動作影片說明
- ⏰ **提醒功能** - 定時提醒進行復健
- 📱 **響應式設計** - 手機、平板、電腦皆可使用

## 復健項目

### 1. 基礎運動
- 臉部運動
- 舌頭運動
- 嘴唇運動
- 咀嚼運動

### 2. 吞嚥練習
- Masako 訓練
- Mendelsohn 手法
- 彭克吞嚥法
- 努德森吞嚥法

### 3. 進食安全
- 正確進食姿勢
- 食物質地選擇
- 進食環境設定

## 技術棧

- React + Vite
- Tailwind CSS
- TypeScript
- 語音/視訊整合（可選）

## 部署

可部署至 Zeabur、Vercel、Netlify 等平台。

## 開發

```bash
npm install
npm run dev
```
